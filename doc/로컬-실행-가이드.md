# 로컬 실행 가이드

## 1. 목적
- `collyapp`를 로컬에서 안정적으로 실행하기 위한 표준 절차를 정의한다.

## 2. 사전 준비
### 공통
- Node.js 설치
- npm 설치
- 프로젝트 경로: `/Users/lee/IdeaProjects/collyapp`

### iOS
- Xcode 설치 (App Store)
- iOS Simulator 컴포넌트 설치
- CocoaPods 설치

### Android
- Android Studio 설치
- Android SDK / Platform-Tools / Build-Tools / Emulator 설치
- Android Virtual Device(AVD) 1개 이상 생성

## 3. 최초 1회 설정
### 3.1 iOS 개발 도구 경로 설정
```bash
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
sudo xcodebuild -license accept
xcodebuild -version
```

### 3.2 iOS Pod 설치
```bash
cd /Users/lee/IdeaProjects/collyapp/ios
bundle exec pod install
cd ..
```

## 4. 실행 방법
### 4.1 Metro 서버 실행 (필수)
```bash
cd /Users/lee/IdeaProjects/collyapp
npm start
```

### 4.2 iOS 실행
- 새 터미널에서 실행:
```bash
cd /Users/lee/IdeaProjects/collyapp
npx react-native run-ios
```

### 4.3 Android 실행
- 에뮬레이터를 먼저 켠 후, 새 터미널에서 실행:
```bash
cd /Users/lee/IdeaProjects/collyapp
npx react-native run-android
```

## 5. 환경값 규칙
- 환경 설정 파일: `src/shared/config/env.ts`
- `dev` 환경 기본 API 주소
  - iOS: `http://localhost:18080`
  - Android Emulator: `http://10.0.2.2:18080`

## 6. 일상 개발 루프
1. 터미널1: `npm start`
2. 터미널2: `npx react-native run-ios` (또는 `run-android`)
3. 코드 수정 후 자동 반영(Fast Refresh)

## 7. 플랫폼별 정책
- 현재 팀 우선 기준: iOS 우선 개발
- Android는 주기적으로 레이아웃/기능 동작 확인
