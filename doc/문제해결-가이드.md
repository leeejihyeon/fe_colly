# 문제해결 가이드

## 1. iOS 실행 시 `xcodebuild requires Xcode` 오류
### 증상
- `xcode-select: error: tool 'xcodebuild' requires Xcode, but active developer directory ...`

### 원인
- Xcode 앱이 미설치이거나, 개발자 경로가 CommandLineTools로 설정됨.

### 해결
```bash
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
sudo xcodebuild -license accept
cd /Users/lee/IdeaProjects/collyapp/ios
pod install
```

## 2. iOS 시뮬레이터가 안 뜨는 경우
### 원인
- Xcode의 iOS Simulator 컴포넌트 미설치.

### 해결
- Xcode 실행 후 iOS Simulator 다운로드/설치 완료
- 이후 `npx react-native run-ios` 재실행

## 3. `A dev server is already running on 8081`
### 의미
- Metro가 이미 실행 중이라는 정상 안내 메시지일 수 있음.

### 조치
- 기존 `npm start` 터미널을 유지한 채, 새 터미널에서 `run-ios`/`run-android` 실행

## 4. Android 디바이스가 안 잡히는 경우
### 확인
```bash
adb devices
```
- `emulator-xxxx device`가 보여야 정상.

### 해결
- Android 에뮬레이터 재실행
- Android Studio Device Manager에서 AVD 상태 확인

## 5. Pod 관련 오류
### 해결 순서
```bash
cd /Users/lee/IdeaProjects/collyapp/ios
pod repo update
pod install
```
